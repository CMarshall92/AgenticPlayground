# Broad Market Research Pipeline

This folder contains a separate discovery-first agent pipeline.

It is intended for broad market scanning and idea discovery before any portfolio sizing or rebalance logic is applied.

Recommended execution order:

1. `01_macro_regime_scanner.md`
2. `02_market_universe_discovery.md`
3. `03_equity_shortlist_validator.md`
4. `04_portfolio_constructor.md`

Use this version when the goal is to start from the market and discover candidates, rather than begin with a predefined ticker list.
